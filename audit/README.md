# Audit — real TestNet trust-lifecycle evidence

Real, confirmed Algorand **TestNet** transactions covering the full ARC-8004 trust lifecycle for one
agent: **register → pay → review → validate**. Generated by `scripts/audit-e2e-testnet.ts`.

- **[LATEST.md](LATEST.md)** — human-readable trail of the most recent run (explorer links per tx).
- `e2e-testnet-<timestamp>.{json,md}` — one immutable snapshot per run. The `.json` is machine-readable
  (every txid, confirmed round, sender, method, args summary).

Each run uses three distinct accounts — **owner/operator**, **paying client/reviewer**, **independent
validator** — so the trail is honest: a client reviews an agent it actually paid (the `giveFeedback`
review is bound to the real x402 settlement txid), and a separate account records the validation verdict.
