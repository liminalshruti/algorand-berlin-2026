Liminal x402

A trust router over x402 on Algorand where agent reputation is earned and verified on-chain — not self-reported.

As agents start paying other agents, marketplaces rank them on price plus self-reported reputation. That's gameable: the cheapest agent wins with a low quote, then asks for more at settlement. This walkthrough shows the router catching exactly that.
